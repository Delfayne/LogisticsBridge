# LogisticsBridge

An add-on mod for Logistics Pipes, AppliedEnergistics 2 EL and Refined Storage.  

To run debug add the following to the JVM args: 
```shell
-Dfml.coreMods.load=logisticspipes.asm.LogisticsPipesCoreLoader
```
